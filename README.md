#  AI Study Assistant

An AI-powered study tool that helps students learn smarter by:
- Summarizing lecture notes
- Generating flashcards
- Running interactive quizzes

## Features (Planned)
- Upload text or PDF notes
- Summarize content using AI (BART or T5 model)
- Automatically create Q&A flashcards
- Quiz mode with spaced repetition

##  Built With
- Python
- Streamlit (for UI)
- HuggingFace Transformers
- SQLite (for user data)

##  Getting Started
1. Clone the repo:
git clone https://github.com/YOUR_USERNAME/ai_study_assistant.git

2. Install dependencies:
pip install -r requirements.txt

3. Run:
streamlit run ui/app.py
